## 💻 概要

- トレーニングアプリ「トレログ」を Next.js と Hasura で作りました。トレーニングや体重などの記録、編集、削除などができます。他にも、体重などの記録をグラフ表示、トレーニングなどの記録や予定をカレンダーから確認することができます 💪

## 🎧 フロントエンド

- Typescript
- Emotion
- GraphQL(apollo/client)
- Next.js

## 🏀 バックエンド

- Hasura(GraphQL Server)

## ⭐️ インフラ

- Vercel(フロントエンド)
- Heroku(バックエンド)

## 💣 DB

- PostgreSQL

## 🔑 認証
